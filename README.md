## рџ“Њ Overview

**Vibhag Setu** is a Faculty Administration System developed to digitize and streamline departmental management in educational institutions. The platform minimizes manual paperwork by providing role-based access for **Admin**, **Faculty**, and **Class Representatives (CR)**.

It enables efficient management of faculty records, attendance, notices, timetables, faculty ID cards, and AI-assisted support through a centralized web application.
